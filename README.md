# 汤原农文旅云认养平台 V1.0

## 项目简介
汤原农文旅云认养平台是一个H5前端项目，为用户在消费帮扶平台购买产品后提供云认养服务。

## 技术栈
- Vue 3 + TypeScript
- Vite
- Element Plus

## 功能模块
1. 登录模块（移动认证/短信验证码）
2. 云认养流程（活动介绍、身份验证、分配田地）
3. 数字田地凭证（展示、溯源、物联网数据）
4. 权益中心
